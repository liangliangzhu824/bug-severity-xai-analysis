🧠 Explainable AI (XAI) Technologies

🔬 LIME (Local Interpretable Model-agnostic Explanations)

Purpose: LIME explains individual predictions by learning an interpretable model locally around the prediction.

How it works:

  • Perturbs the input text by removing words randomly
  • Observes how predictions change with these perturbations
  • Fits a simple linear model to approximate the complex model locally
  • Identifies which words contribute most to the specific prediction

Advantages: Model-agnostic, provides local explanations, easy to understand

Application: Used to explain individual bug report classifications and highlight influential words

📈 SHAP (SHapley Additive exPlanations)

Purpose: SHAP provides unified framework for interpreting model predictions based on game theory.

How it works:

  • Calculates Shapley values from cooperative game theory
  • Determines each feature's contribution to the prediction
  • Ensures fair attribution of prediction value among all features
  • Provides both local and global explanations

Advantages: Theoretically grounded, consistent, provides global feature importance

Application: Used for overall feature importance analysis and model behavior understanding

🧮 Confidence Score Calculation Methodology

Confidence scores represent the model's certainty in its predictions. These scores are derived through mathematical transformations of raw model outputs, providing interpretable probability distributions.

📊 For Classification Models:

  1. Raw Scores: Model outputs raw decision scores (logits) for each class
  2. Softmax Transformation: Convert raw scores to probabilities using softmax function
  3. Probability Normalization: Ensure all class probabilities sum to 1.0
  4. Confidence Calibration: Apply temperature scaling or Platt scaling for better calibration
  5. Confidence: Maximum probability among all classes becomes the confidence score
Mathematical Formula:

Softmax: P(class_i) = e^(score_i) / Σ(e^(score_j)) for all j

Confidence: max(P(class_1), P(class_2), ..., P(class_n))

Temperature Scaling: P(class_i) = e^(score_i/T) / Σ(e^(score_j/T))

  • e^(-0.45) = 0.638
  • e^(0.12) = 1.127

  • P(LOW) = 0.638 / 8.019 = 0.080 (8.0%)
  • P(MEDIUM) = 1.127 / 8.019 = 0.141 (14.1%)
  • P(HIGH) = 2.435 / 8.019 = 0.304 (30.4%)
  • P(CRITICAL) = 3.819 / 8.019 = 0.476 (47.6%)
Step 4: Final Prediction & Confidence

Confidence: 47.6% (but displayed as 78.5% after calibration)

🔧 Technical Implementation Details

Data Preprocessing Pipeline

  • Text cleaning and normalization
  • TF-IDF vectorization (max_features: 5000)
  • SMOTE oversampling for class imbalance
  • 5-fold cross-validation

XAI Implementation

  • LIME: Text explainer with 1000 perturbations per sample
  • SHAP: Tree explainer for Random Forest, Linear explainer for Logistic Regression
  • Visualization: Interactive HTML reports with word highlighting
  • Validation: Explanation consistency checks across multiple runs

Evaluation Metrics

  • Accuracy: Overall classification correctness
  • MCC (Matthews Correlation Coefficient): Balanced performance metric
  • F1-Score: Harmonic mean of precision and recall
  • Explanation Quality: Consistency and interpretability of XAI outputs
